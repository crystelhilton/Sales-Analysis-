# Sales Data Analysis

## Overview
This project analyzes sales data from a CSV file and provides insights on total sales, category-wise distribution, top products, and region-wise performance.

## Purpose of Git Workflow

To execute these Git commands in VS Code: Open your `analysis.py` file in VS Code, then open the integrated terminal by pressing `Ctrl+`` (backtick). This opens a command prompt at the bottom of VS Code in your project directory. You then type each Git command one at a time and press Enter to execute it. For example, type `git init` and press Enter to initialize a repository, then `git add analysis.py` and press Enter to stage your file, followed by `git commit -m "Initial analysis using Copilot"` and press Enter to create your first commit. Each command executes sequentially, with the terminal showing confirmation messages after each one. You can view your commit history by typing `git log --oneline` to verify both commits were created successfully.
